# Maguro to Shari v2

Lightweight website framework built outside Hostinger first for later migration.

## Principles
- Portable: plain HTML/CSS/JS
- Mobile-first
- Bilingual-ready (JP/EN)
- Reservation flow is request-based, not auto-confirmed
- Walk-in first, premium bowl pre-order reservation only

## Initial pages
- `index.html` — homepage
- `menu.html` — menu
- `reserve.html` — reservation / pre-order
- `thank-you.html` — pending confirmation page

## Files
- `assets/css/styles.css`
- `assets/js/site.js`
- `assets/js/reserve.js`
- `data/site-content.js` — easy-to-update content/config
- `docs/framework-plan.md` — structure, layout, form logic, missing materials

## Notes
This first pass focuses on framework, structure, layout, and booking flow only.
No polish, no final copy, no final imagery.
